This repository contains the data and scripts used to conduct the genomic and phenotypic analyses for this study. The project is organized to clearly separate input data, analysis code, and derived outputs, ensuring transparency, reproducibility, and ease of reuse.
## Repository structure

- **SNP_fil**  
  Contains single nucleotide polymorphism (SNP) marker data for all genotypes included in the study.

- **BLUE_fil**  
  Contains best linear unbiased estimates (BLUEs) for each genotype used in downstream analyses.

- **Scripts_repo**  
  Contains the complete, end-to-end analysis pipeline required to reproduce the results.
